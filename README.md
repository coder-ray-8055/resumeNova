# Auto Resume Builder

Auto Resume Builder is a simple project that helps users generate professional resumes quickly by providing their details.

This project is currently under development and new features will be added gradually.

## Features (Planned)
- Input personal details
- Generate formatted resume
- Download resume
- Clean UI
- Multiple resume templates (planned)
