<h1 align="center">🚀 Auto Resume Builder</h1>

<p align="center">
Create professional resumes instantly by entering your personal and professional details.
</p>

<p align="center">
A simple and fast resume generator built using HTML, CSS and JavaScript.
</p>

---

## ✨ Features

✔️ Easy input form for user details  
✔️ Instant resume generation  
✔️ Profile photo upload support  
✔️ Clean and modern UI  
✔️ Download generated resume  
✔️ Multiple resume templates
